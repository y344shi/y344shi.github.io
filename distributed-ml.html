Project Overview

This project developed research prototypes to explore distributed training strategies by combining model, data, and pipeline parallelism. The focus was on capturing execution traces to build scalable execution graphs, enabling deeper insight into optimizing distributed machine learning workflows.

Key Features

  • Hybrid Parallelism Techniques
  • Execution Trace Capture & Analysis
  • Scalable Graph Construction
  • Data Visualization with Matplotlib

Technologies Used

  • Python & PyTorch
  • Matplotlib
  • Distributed Systems

The project identified bottlenecks in distributed ML systems and proposed optimizations that led to significant performance improvements.
